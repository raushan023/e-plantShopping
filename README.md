e-plantShopping 🌿
Welcome to e-plantShopping, a modern React application for Paradise Nursery. This project is a dedicated online platform for house plant enthusiasts, designed to make browsing and purchasing greenery a breeze.

✨ Key Features
🪴 Product Management
Categorized Sections: Plants are organized into distinct groups like "Aromatic Plants" and "Medicinal Plants" for easier navigation.

Detailed Plant Cards: Each plant features a high-quality image, name, description, cost, and an Add to Cart button.

🛒 Shopping Cart Functionality
Dynamic Cart Updates: A navigation bar that tracks the number of items in your cart across all pages.

Quantity Controls: Within the cart, users can increase or decrease quantities or remove items entirely.

Automatic Calculations: The cart automatically calculates the cost for all plants of a specific type as well as the grand total.

🛠️ Tech Stack & Requirements
This repository, e-plantShopping, is built using:

React.js (Frontend Library)

Redux Toolkit (State Management)

CSS3 (Responsive Styling)

🚀 Installation & Setup
To run the e-plantShopping project locally, follow these steps:

1. Clone the repository:
git clone https://github.com/raushan023/e-plantShopping.git

2. Navigate into the directory:
cd e-plantShopping

3. Install dependencies:
npm install

4. start in dev env:
npm run dev
Open http://localhost:5173 to view it in your browser.

📁 Project Structure
Landing Page: The entry point with a "Get Started" call to action.

Product Listing: The main gallery featuring the plant categories.

Cart Page: The management hub for reviewing items, adjusting quantities, and checking out.
